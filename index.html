Minecraft: conheça as diferentes versões do jogo e seus detalhes

Minecraft (PS Vita Edition)

O Minecraft Java Edition é a versão que estamos mais acostumados a ter contato atualmente, sendo popularmente chamada de "Minecraft para PC". Esta é a versão mais comum encontrada em nossos servidores e que tem o melhor suporte para mods, plugins e customizações em geral.

Minecraft (Bedrock)

“Então, como essa versão difere do Minecraft que já posso jogar no meu PS4?” A diferença é que a versão Bedrock de Minecraft é a versão unificada do jogo, que oferece a mesma experiência através de todos os consoles.

Qual é a diferença entre Minecraft Java e Bedrock?

*O Minecraft: Java Edition roda no Windows, no Mac e no Linux. O Minecraft: Bedrock Edition roda no Windows. O conteúdo da Deluxe Collection roda apenas no Minecraft: Bedrock Edition para Windows.
